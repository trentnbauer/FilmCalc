Calculator Settings

Currency

Symbol shown in front of every cost throughout the app. This just changes the label — it doesn't convert any numbers.

Recommended Pick Threshold

In "Labs For This Roll", if the cheapest Hi-Res + Fastest option costs less than this percentage more than the absolute cheapest, its card is highlighted gold as the recommended pick — since a small premium is usually worth both perks. "Cheapest Total" always shows the actual cheapest option, unchanged.

Data

Export Data

Download your saved films/labs/settings as YAML. "Export Everything" writes a config.yaml. "Films/Labs Only" asks for a preset name that's saved inside the file — so it shows up by that name when re-imported (drag & drop supported) or dropped into the project's films/ or labs/ folders.

Import Data

Pick from the project's built-in film/lab presets, or upload your own config.yaml, films.yaml, or labs.yaml.

Write Live Config to Server

Self-hosted only. Overwrites config.yaml on the server directly with your current saved films, labs, and settings — no download/upload round-trip needed.

Delete All Data

Permanently deletes every saved film, lab, and setting from this browser. Built-in defaults from config.yaml aren't affected. Consider exporting a backup first — this can't be undone.
